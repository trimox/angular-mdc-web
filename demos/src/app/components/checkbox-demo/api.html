MdcCheckbox

Selector: mdc-checkbox

Exported as: mdcCheckbox

Properties

Name Description
id: string Unique Id of the checkbox (auto generated if not supplied).
name: string Name of the checkbox.
value: string Value of the checkbox.
tabIndex: number Set the underlying tab index of the checkbox. (Default: 0)
ariaLabel: string Used to set the 'aria-label' attribute on the underlying input element.
ariaLabelledby: string The 'aria-labelledby' attribute takes precedence as the element's text alternative.
checked: boolean Sets the checkbox input to checked or not.
indeterminate: boolean Represent a checkbox with three states (e.g. a nested list of checkable items).
disabled: boolean Disables the component.
disableRipple: boolean Whether ripple ink is disabled.
indeterminateToChecked: boolean Whether the checkbox should go to checked state or unchecked when toggled from indeterminate state.

Methods

focus() Set focus to the checkbox.
toggle(checked?: boolean) Toggles checkbox via user action. When it is indeterminate, toggle can go to checked or unchecked, depending on state.

Events

change(source: MdcCheckBox, value: boolean) Event dispatched on value change.
indeterminateChange(source: MdcCheckbox, indeterminate: boolean) Emit when checkbox goes in and out of indeterminate state, but not when set to checked.
