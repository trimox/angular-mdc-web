Design & API Documentation

mdc-icon

Properties Description
fontSet: string Font set that the icon is a part of.
fontIcon: string Name of an icon within a font set.
fontSize: number The size of the icon. If no size is specified, the default of 24px is used.
clickable: boolean Whether or not icon can be interacted with.
Methods Description
setClickable(clickable: boolean) Whether or not the icon is clickable.
isLeading(): boolean Whether or not the icon is set as leading icon.
isTrailing(): boolean Whether or not the icon is set as trailing icon.
