MdcChipSet

Selector: mdc-chip-set

Exported as: mdcChipSet

Properties

Name Description
choice: boolean Indicates that the chips in the set are choice chips, which allow a single selection from a set of options.
filter: boolean Indicates that the chips in the set are filter chips, which allow multiple selection from a set of options.
input: boolean Indicates that the chips in the set are input chips, which enable user input by converting text into chips.

Methods

getSelectedChipIds(): string[] Returns an array of the IDs of all selected chips.
select(chipId: string): void Selects the chip with the given id. Deselects all other chips if the chip set is of the choice variant.

Events

interaction: MdcChipInteractionEvent Indicates when a chip is interacted with (via click/tap or Enter key)
change: MdcChipSetChange Emitted when a chip is interacted with.

MdcChip

Selector: mdc-chip

Exported as: mdcChip

Properties

Name Description
choice: boolean Indicates that the chips in the set are choice chips, which allow a single selection from a set of options.
filter: boolean Indicates that the chips in the set are filter chips, which allow multiple selection from a set of options.
input: boolean Indicates that the chips in the set are input chips, which enable user input by converting text into chips.
label: string Sets the text content of the chip.
value: string | string[] The value of the chip. Defaults to the content inside mdc-chip.
removable: boolean Sets whether a trailing icon click should trigger exit/removal of the chip. (Default is true)
disableRipple: boolean Whether ripples are disabled.

Methods

focus() Set focus to the chip.

Events

interactionEvent: MdcChipInteractionEvent Indicates the chip was interacted with (via click/tap or Enter key)
selectionChange: MdcChipSelectionEvent Indicates the chip's selection state has changed (for choice/filter chips)
removalEvent: MdcChipRemovalEvent Indicates the chip removal event from a chip set
navigationEvent: MdcChipNavigationEvent Indicates a navigation event has occurred on a chip
trailingIconInteraction: MdcChipInteractionEvent Indicates the chip's trailing icon was interacted with (via click/tap or Enter key)

MdcChipText

Optional. Indicates the text content of the chip.

Selector: mdc-chip-text

Exported as: mdcChipText

MdcChipIcon

Optional. Indicates the text content of the chip.

Selector: mdc-chip-icon

Exported as: mdcChipIcon

Properties

Name Description
leading: boolean Indicates a leading icon in the chip
trailing: boolean Indicates a trailing icon in the chip
