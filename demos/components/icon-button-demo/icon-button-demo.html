Icon Buttons

Icon buttons allow users to take actions, and make choices, with a single tap.

References

  • Material Design guidelines: Toggle Buttons
  • Material Components Web

Import

Design & API Documentation

button[mdcIconButton]

a[mdcIconButton]

Properties Description
on: boolean Sets the toggle state to the provided value.
iconOn: string Set icon as content when toggled on.
iconOff: string Set icon as content when toggled off.
labelOn: string Set "aria-label" attribute when toggled on.
labelOff: string Set "aria-label" attribute when toggled off.
primary: boolean Colors the button with the primary theme color.
secondary: boolean Colors the button with the secondary theme color.
disabled: boolean Disables the icon button.
Methods Parameters
toggle(isOn: boolean) Toggles the icon toggle state. If an argument is given, will toggle on if true, off if false.
refreshToggleData() Allow's developers to refresh toggle data if needed.
isOn() Returns true if the button is toggled on, false otherwise.
Events Description
change: MdcIconButtonChange Emits when the icon is toggled.
