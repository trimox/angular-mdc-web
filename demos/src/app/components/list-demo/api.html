MdcList

Selector: mdc-list

Exported as: mdcList

Properties

Name Description
dense: boolean Increases the density of the list, making it appear more compact.
twoLine: boolean Increases the height of the row to give it greater visual separation from adjacent rows.
avatar: boolean Configures the leading tiles of each row to display images instead of icons. This will make the graphics of the list items larger.
border: boolean Sets a border around each list item.
singleSelection: boolean When enabled, the space and enter keys (or click event) will trigger an single list item to become selected and any other previous selected element to become deselected.
interactive: boolean List items receive styles for hover, focus, and press states (including the ripple). Default is true.
useActivatedClass: boolean Sets the selection logic to apply/remove the mdc-list-item--activated class.
useSelectedClass: boolean Sets the selection logic to apply/remove the mdc-list-item--selected class.
wrapFocus: boolean Sets the list to allow the up arrow on the first element to focus the last element of the list and vice versa.
verticalOrientation: boolean Sets the list to an orientation causing the keys used for navigation to change. true results in the Up/Down arrow keys being used. If false, the Left/Right arrow keys are used.
disableRipple: boolean Disables the ripple effect.

Methods

setSelectedValue(value: any) Sets the currently selected list item by value.
setSelectedIndex(index: number) Sets the currently selected list item.
getSelectedIndex(): number Returns the currently selected item index.
getSelectedItem(): MdcListItem | undefined Returns the currently selected item.
focusItemAtIndex(index: number) Sets focus to the element at the given index in a list.
focusFirstElement() Sets focus to the first element in a list.
focusLastElement() Sets focus to the last element in a list.
getListItemByValue(value: any): MdcListItem | undefined Returns list item for given value.
getListItemByIndex(index: number): MdcListItem | undefined Returns list item for given index.
getListItemIndexByValue(value: any): number Returns the list item index for a given value.
reset() Clear all activated and selected list item states.

Events

selectionChange: MdcListItemChange Change event that is being fired whenever the selected state of an option changes.

MdcListItem

Selector: mdc-list-item a[mdc-list-item]

Exported as: mdcListItem

Properties

Name Description
selected: boolean Styles the row in an selected state.
activated: boolean Styles the row in an activated state.
disabled: boolean Disables the list item.

MdcListItemGraphic

Optional. the first tile in the row (in LTR languages, the first column of the list item). Typically an icon or image.

Selector: mdcListItemGraphic

Exported as: mdcListItemGraphic

MdcListItemMeta

Optional. the last tile in the row (in LTR languages, the last column of the list item). Typically small text, icon. or image.

Selector: mdcListItemMeta

Exported as: mdcListItemMeta

MdcListItemText

Mandatory. Wrapper for list item text content (displayed as middle column of the list item).

Selector: mdc-list-item-text mdcListItemText

Exported as: mdcListItemText

Properties

Name Description
secondaryText: string Optional secondary text for the list item. Displayed below the primary text.

MdcListItemSecondary

Optional. Secondary text for the list item. Displayed below the primary text. Should be the child of mdc-list-item-text.

Selector: mdc-list-item-secondary mdcListItemSecondary

Exported as: mdcListItemSecondary

MdcListDivider

Optional, for list divider element.

Exported as: mdcListDivider
