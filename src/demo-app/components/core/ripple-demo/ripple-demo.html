Ripple

MDC Ripple provides components (or any element) with a material "ink ripple" interaction effect.
Directive
mdc-ripple
Properties Description
ripple: MdcRippleOrchestration Use ripple orchestration properties and methods.
active: boolean Activate or de-activate ripple ink effect. (Default is true)
primary: boolean Colors the ripple with the primary theme color.
secondary: boolean Colors the ripple with the secondary theme color.
