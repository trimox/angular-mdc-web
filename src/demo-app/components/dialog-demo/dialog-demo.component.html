Dialog

Dialogs inform users about a specific task, require decisions, or involve multiple tasks.
Component
mdc-dialog The main component which is composed of mdc-dialog elements.
Methods Parameters
open() Opens the dialog, registers appropriate event listeners, sets aria attributes, focuses elements.
isOpen() Returns true if the dialog is open, false otherwise.
close() Closes the dialog, deregisters appropriate event listeners, resets aria attributes, focuses elements.
Events Description
accept Broadcast when a user accepts the dialog.
cancel Broadcast when a user cancels the dialog.
Sample - Dialog 
<mdc-dialog #dialog>
  <mdc-dialog-header>
    <mdc-dialog-header-title>
      Use Google's location service?
    </mdc-dialog-header-title>
  </mdc-dialog-header>
  <mdc-dialog-body>
    Let Google help apps determine location. This means sending anonymous location data to Google, even when no apps are running.
  </mdc-dialog-body>
  <mdc-dialog-footer>
    <button mdc-button mdc-dialog-button-cancel (click)="cancelDialog();">Decline</button>
    <button mdc-button mdc-dialog-button-accept (click)="acceptDialog();">Accept</button>
  </mdc-dialog-footer>
</mdc-dialog>
