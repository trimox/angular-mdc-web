Slider

Sliders allow users to make selections from a range of values.

  • Material Design guidelines: Sliders

Import

Design & API Documentation

Selector: mdc-slider

Properties Description
min: number The minimum value that the slider can have.
max: number The maximum value that the slider can have.
value: number The current value of the slider.
step: number Specifies the increments at which a slider value can be set.
discrete: boolean Discrete sliders allow users to select a specific value from a range. (Default: false)
markers: boolean Show markers on track. Discrete sliders support displaying markers on their tracks. (Default: false)
disabled: boolean Disables the toggle slider.
Methods Parameters
getMin(): number Returns the min value the slider can have.
getMax(): number Returns the max value the slider can have.
getStep(): number Returns the slider step value.
getValue(): number Returns the current value of the slider.
Events Description
change(source: MdcSlider, value: number) Broadcast when slider value is changed and committed by way of a user event, e.g. when a user stops dragging the slider or changes the value using the arrow keys.
input(source: MdcSlider, value: number) Broadcasts when slider value is changed by way of a user event, e.g. when a user is dragging the slider or changing the value using the arrow keys.
