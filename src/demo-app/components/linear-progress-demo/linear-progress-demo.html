Linear Progress

The MDC Linear Progress component is a spec-aligned linear progress indicator component adhering to the Material Design progress & activity requirements.
Component
mdc-linear-progress
Properties Description
indeterminate: boolean Puts the linear progress indicator in an indeterminate state.
reversed: boolean Reverses the direction of the linear progress indicator.
secondary: boolean Colors the button with the secondary theme color.
closed: boolean Hides the linear progress indicator.
Methods Parameters
open() Puts the component in the open state.
close() Puts the component in the closed state.
setProgress(value: number) Sets the progress bar to this value. Value should be between [0, 1].
setBuffer(value: number) Sets the buffer bar to this value. Value should be between [0, 1].
setReverse(value: boolean) Reverses the direction of the linear progress indicator.
setDeterminate(value: boolean) Toggles the component between the determinate and indeterminate state.
