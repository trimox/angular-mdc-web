Chips

Chips are compact elements that allow users to enter information, select a choice, filter content, or trigger an action.

Design & API Documentation

Selector: mdc-chip-set

Properties Description
choice: boolean Indicates that the chips in the set are choice chips, which allow a single selection from a set of options.
filter: boolean Indicates that the chips in the set are filter chips, which allow multiple selection from a set of options.
input: boolean Indicates that the chips in the set are input chips, which enable user input by converting text into chips.
Methods Description
getSelectedChipIds(): string[] Returns an array of the IDs of all selected chips.
select(chipId: string): void Selects the chip with the given id. Deselects all other chips if the chip set is of the choice variant.
deselect(chipId: string): void Deselects the chip with the given id.
Events Description
change: MdcChipSetChange Emitted when a chip is interacted with.

Selector: mdc-chip

Properties Description
label: string Sets the text content of the chip.
removable: boolean Sets whether a trailing icon click should trigger exit/removal of the chip. (Default is true)
primary: boolean Colors selected chip with the primary theme color.
secondary: boolean Colors selected chips with the secondary theme color.
Events Description
selectionChange: MdcChipInteractionEvent Emitted when the chip is selected or deselected.
removed: MdcChipRemovedEvent Emitted when a chip is removed from a set.

Selector: mdc-chip-text

Optional. Indicates the text content of the chip.

Selector: mdc-chip-icon

Properties Description
leading: boolean Indicates a leading icon in the chip
trailing: boolean Indicates a trailing icon in the chip
