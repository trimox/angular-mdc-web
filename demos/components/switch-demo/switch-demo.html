Switches

Switches toggle the state of a single setting on or off.

Design & API Documentation

mdc-switch

Properties Description
id: string Unique Id of the switch button (auto generated if not supplied).
name: string Name of the switch.
ariaLabel: string Used to set the 'aria-label' attribute on the underlying input element.
ariaLabelledby: string The 'aria-labelledby' attribute takes precedence as the element's text alternative.
tabIndex: number Set the underlying tab index of the switch. (Default: 0)
checked: boolean Use to set the checked value.
disabled: boolean Disables the component.
Methods Description
focus() Set focus to the switch.
Event Description
change(source: MdcSwitch, value: boolean) Event dispatched on value change.
