Getting started

For help getting started with a new Angular app, check out the Angular CLI.

For existing apps, follow this guide to use Angular MDC.

Install Angular MDC

npm

yarn

Roboto font

To get started, first include the Roboto font with the 300, 400 and 500 weights. You can host it yourself or include it from Google Fonts:

Material Icons

If you want to use the mdc-icon component with the official Material Design Icons, load the icon font in your index.html:
For more information on using Material Icons, check out the Material Icons Guide. Note that mdc-icon supports any font or svg icons; using Material Icons is one of many options.

MDC Sass modules

In your angular.json, make sure node_modules/ is listed as a Sass include path. This is needed for the Sass compiler to be able to find the MDC Web Sass files.

Import Angular MDC components

Create module file `./app/material.module.ts`, and import the NgModule for each component you want to use.
