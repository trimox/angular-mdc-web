Top App Bar

The top app bar displays information and actions relating to the current screen.

References

  • Material Design guidelines: Top App Bar
  • Material Components Web

Import

Design & API Documentation

Selector: mdc-top-app-bar

Properties Description
fixed: boolean Style the app bar as a fixed top app bar.
short: boolean Style the app bar as a short app bar. Short top app bars should only be used with one action item.
prominent: boolean Enables prominent variant.
dense: boolean Enables dense variant.
shortCollapsed: boolean Set the short app bar to always collapsed.
fixedAdjustElement: any If set, scroll event listener is attached to element object.
scrollTarget: target: Element | Window Sets scroll target to different DOM node (default is window).
Methods Description
isCollapsed: boolean Whether or not the app bar is in a collapsed state.
Events Description
navigationSelected: MdcTopAppBarNavSelected Emitted when the top app bar navigation menu icon is selected.

Selector: mdc-top-app-bar-row mdcTopAppBarRow

A one-line row to add app bar sections.

Selector: mdc-top-app-bar-section mdcTopAppBarSection

A content area for navigation or actions.
Properties Description
title: string Optional. The text used to describe the app or section.
align: string = 'start' | 'end' Align section to either 'start' or 'end' of row.

Selector: mdc-top-app-bar-title mdcTopAppBarTitle

Optional. The text used to describe the app or section.

Selector: mdc-icon[mdcTopAppBarNavIcon]

Left most icon of app bar used for navigation.

Selector: mdc-icon[mdcTopAppBarActionItem]

Top app bars can accommodate multiple action items on opposite side of navigation icon.
